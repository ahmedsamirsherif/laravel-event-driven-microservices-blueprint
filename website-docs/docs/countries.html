Country Modules

The platform uses an auto-discovering convention-based architecture. Adding a new country requires zero changes to existing code — just create files in the right directories.

Supported Countries

🇺🇸 USA SSN + Address required 2 extra fields, 3 checklist steps
🇩🇪 DEU (Germany) Tax ID + Goal required 2 extra fields, 3 checklist steps

Adding a New Country

  1. Add case to CountryCode enum (both services)
  2. Create Domain/Country/{ISO3}/{ISO3}Fields.php implementing CountryFieldsInterface (HR)
  3. Create Domain/Country/{ISO3}/{ISO3}Module.php implementing CountryModuleInterface (Hub)
  4. Update event schema contract (if new fields)
  5. Add tests — done!
